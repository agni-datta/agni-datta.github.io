Privacy

This site collects the minimum information needed to remember your preferences and understand coarse traffic patterns.

What the site stores

When you choose a color theme, the site stores a first-party theme cookie for one year. The cookie contains only light or dark and is used only to preserve that functional preference.

Cloudflare provides the country and region associated with a successful HTML page request. The site adds one page view to a daily aggregate for a bounded page category, country code, and region code. Missing location values become unknown.

The analytics system does not read or retain IP addresses, forwarded IP headers, city, coordinates, postal code, user agent, referrer, cookies, or a persistent visitor identifier. It measures page views, not unique people, and introduces no analytics cookie or fingerprinting.

Aggregate rows are deleted automatically after 90 days. Cloudflare provides the edge and database services, while GitHub Pages hosts the site origin. To ask a privacy question, email {{ data.person.email_display }}.
